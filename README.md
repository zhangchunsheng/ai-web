# 烙馍网 AI 团队站点

烙馍网（Luomor.com）AI 团队的静态品牌展示站点集合。

## 项目简介

烙馍网 AI 团队是一个由独立开发者驱动的个人 AI 公司品牌站。本项目包含多个不同主题风格的着陆页，每个页面都是完全独立的 HTML 文件，带有内联 CSS 和第三方统计/广告脚本。

## 页面列表

- `index.html` — 深色科幻风 · 一人 AI 公司概念页
- `about.html` — 企业蓝 · AI 团队介绍页
- `gemini.html` — 橙色务实风 · AI 落地实施页
- `grok.html` — 红色传统风 · 烙馍饮食文化页
- `luomor_ai_team.html` — 金色高端风 · AI 团队企业页

## 技术栈

纯静态 HTML + 内联 CSS + 第三方 JavaScript（Google AdSense / Analytics / 百度统计）。无构建工具，无框架。

## 开发

直接编辑 HTML 文件，在浏览器中打开即可预览。
